# Pour Bowy

Mod Forge 1.6.4 ajoutant plusieurs nouveaux blocs de terre décoratifs.

## Construction

1. Installez Java 7 ou 8 ainsi que Gradle 1.12.
2. Exécutez `gradle setupDevWorkspace` pour préparer l'environnement.
3. Exécutez `gradle build` pour générer le fichier JAR dans `build/libs`.
4. Copiez le JAR dans le dossier `mods` d'une installation Minecraft 1.6.4 utilisant Forge 9.11.1.1345.

## Blocs

Les blocs suivants sont ajoutés, avec des identifiants configurables (par défaut 3000-3009) :

- Packed Dirt
- Mossy Dirt
- Rich Dirt
- Dry Dirt
- Fertile Dirt
- Porous Dirt
- Dense Dirt
- Soft Dirt
- Silted Dirt
- Crusted Dirt

Le fichier de configuration `config/pourbowy.cfg` est généré lors du premier lancement et permet de modifier les identifiants (toujours compris entre 3000 et 3500).
